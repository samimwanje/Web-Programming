Fetch Server Quiz

You are going to answer on some questions from a pbulic client application through a RESTful Web API. The first question (starting point of the application) is at the URL: https://courselab.lnu.se/quiz/question/1

At the start of the game, you will be able to write a username and a URL to the RESTful Web API that is going to be used in the quiz game. The game have a timer that gives you the maximum time of 10 seconds to answer each question. If you do not answer during the time or give a false answer, the game is over and you should be able to start over by sending ENTER.

If you answer all of the questions correctly, the game will end and you will recieve a proper "victory message" and showing the time it took to complete the game. A high score list is presented to the right, your username will be included in the list if you manage to answer all the questions in good time.
